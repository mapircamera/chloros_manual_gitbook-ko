## 언어 변경 방법

### Chloros 데스크톱/브라우저에서

1. 애플리케이션 설정 열기
2. 언어 선택 메뉴로 이동
3. 목록에서 원하는 언어 선택
4. 인터페이스가 즉시 업데이트됩니다

### Chloros CLI에서

명령어와 함께 언어 매개변수를 사용하여 인터페이스 언어를 설정하세요.

### Chloros Python SDK

SDK 초기화 시 언어 매개변수를 설정하면 메시지와 출력을 원하는 언어로 확인할 수 있습니다.

## 지원 범위

38개 언어 모두 다음 환경에서 완벽히 지원됩니다:

* **Chloros 데스크톱** - GUI 전체 번역
* **Chloros 브라우저** - 모든 언어의 웹 인터페이스
* **Chloros CLI** - 명령줄 인터페이스 및 출력 메시지
* **Chloros Python SDK** - API 메시지 및 문서

언어 지원은 전 세계 사용자가 모국어로 장벽 없이 효율적으로 작업할 수 있도록 보장합니다.
