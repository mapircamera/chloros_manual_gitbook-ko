# 이미지 그리드

프로젝트에 이미지를 가져온 후 메인 영역에서 그리드 형태로 배열된 이미지를 확인할 수 있습니다.

## 썸네일 크기

오른쪽 상단의 확대/축소 슬라이더를 사용하여 이미지 썸네일 크기를 조정할 수 있습니다. CTRL 키를 누른 상태에서 마우스 휠을 사용해도 크기를 조절할 수 있습니다.

## 내보내기 보기 버튼

이미지 처리 과정에서 그리드 영역 왼쪽 상단에 버튼이 표시됩니다. 이 버튼들은 그리드 썸네일을 다양한 이미지 유형으로 전환합니다:

* **JPG** - 원본 카메라 JPG 썸네일
* **타겟** - 검출된 보정 타겟이 포함된 이미지
* **반사율** - 보정된 반사율 이미지
* **인덱스/LUT** - 컬러 LUT 그라데이션이 적용된 계산된 다중 스펙트럼 인덱스 이미지

이미지 그리드 썸네일을 클릭하면 전체 화면 [이미지 뷰어](opening-an-image-full-screen.md)가 열립니다. 그리드의 보기 모드가 &quot;표적&quot;과 같은 특정 내보내기 유형으로 설정된 경우, 그리드 썸네일을 클릭하면 내보낸 표적 이미지가 전체 화면으로 열립니다.
