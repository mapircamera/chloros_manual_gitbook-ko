# 지도 마커

지도 탭은 GPS 좌표를 기반으로 이미지를 대화형 2D 지도에 표시합니다. 이를 통해 촬영 세션의 지리적 개요를 파악하고 공간적 범위를 시각화할 수 있습니다. 또한 이미지를 처음 가져올 때 처리할 필요가 없는 이미지를 신속하게 제거하는 데 유용합니다.

<figure><img src="../.gitbook/assets/chloros_map_markers.gif" alt=""><figcaption></figcaption></figure>

## 지도 탭 접근 방법

1. Chloros에서 프로젝트 열기 또는 생성
2. GPS 메타데이터가 포함된 이미지 가져오기
3. 왼쪽 사이드바의 **지도** <img src="../.gitbook/assets/image (3).png" alt="" data-size="line"> 탭을 클릭합니다
4. 지도에 각 이미지의 GPS 위치에 마커가 표시됩니다

{% hint style="info" %}
**GPS 필수**: EXIF 메타데이터에 GPS 좌표가 포함된 이미지만 지도에 표시됩니다. 촬영 시 카메라의 GPS 기능이 활성화되었는지 확인하십시오.
{% endhint %}

## 지도 탭에서 이미지 조정하기

**지도** <img src="../.gitbook/assets/image (3).png" alt="" data-size="line"> 탭에는 동일한 추가  <img src="../.gitbook/assets/image.png" alt="" data-size="line">   <img src="../.gitbook/assets/image (1).png" alt="" data-size="line">  추가 및 제거  <img src="../.gitbook/assets/image (2).png" alt="" data-size="line">  파일 버튼을 제공합니다. 또한 동일한 프로젝트 파일 테이블 목록을 표시하지만 열 헤더가 다릅니다:  ### 파일 이름  * 카메라의 원본 파일명 * 카메라의 명명 규칙 유지 (예: IMG\_000001.JPG) <img src="../.gitbook/assets/icon_file-browser.JPG" alt="" data-size="line"> 탭과 동일한 파일 추가및 제거 버튼을 제공합니다. 또한 동일한 프로젝트 파일 테이블 목록을 표시하지만 열 헤더가 다릅니다:

### 파일 이름

* 카메라의 원본 파일 이름
* 카메라의 이름 지정 규칙을 유지합니다(예: IMG\_0001.RAW)

### 위도

* 이미지의 위도

### 경도

* 이미지의 경도

### 고도

* 이미지의 고도

{% hint style="info" %}
테이블 열 헤더를 클릭하면 행 데이터가 정렬됩니다
{% endhint %}

## 이미지 마커

GPS 데이터가 포함된 각 이미지는 지도 상의 마커로 표시됩니다:

### 마커 표시

* 마커는 각 이미지가 촬영된 정확한 GPS 좌표를 표시합니다
* 축소 시 클러스터링된 마커가 그룹화될 수 있습니다
* 확대하여 개별 이미지 위치를 확인하세요

{% hint style="success" %}
초고배율: 지도 타일 제공자의 최대 확대 수준에 도달하면 추가 확대 시 타일이 확대되어 근접한 마커를 확인할 수 있습니다.
{% endhint %}

### 마우스 오버 미리보기

* **마우스를 마커 위에 올려놓으면** 해당 이미지의 축소판 미리보기가 표시됩니다
* 지도 화면을 벗어나지 않고도 빠르게 시각적으로 확인할 수 있습니다
* 대규모 촬영 세션 내에서 특정 이미지를 찾는 데 유용합니다

***

## 지도 타일 제공자

{% hint style="success" %}
**자동 선택**: Chloros는 현재 지도 위치에 최적의 확대/축소 수준을 제공하는 타일 서비스를 자동으로 선택합니다. 필요 시 수동으로 제공자를 전환할 수 있습니다.
{% endhint %}

지도 탭은 배경 지도 이미지에 두 가지 타일 제공자를 지원합니다:

### Google 지도

* Google의 표준 위성 및 지도 이미지
* 전 세계적 커버리지에 최적

### ESRI

* ESRI ArcGIS의 위성 및 항공 이미지
* 특정 지역에서 더 높은 해상도의 이미지를 제공하는 경우가 많음

***

## 지도 타일 유형

지도 레이어 유형을 선택할 수 있습니다(왼쪽부터 오른쪽 순):

 <img src="../.gitbook/assets/image (23).png" alt="" data-size="line">### 지형

고도 프로필과 세부 정보(도로 등)가 포함된 지도 타일을 표시합니다

### 지도

세부 정보(도로 등)가 포함된 표준(낮은 대역폭) 지도 타일을 표시합니다

### 위성

상세한(높은 대역폭) 위성 지도 타일을 표시합니다

### 하이브리드

세부 정보(도로 등)가 추가된 위성 지도 타일을 표시합니다

## 지도 탐색

### 확대/축소 컨트롤

* **확대/축소**: 마우스 스크롤 휠 또는 확대/축소 버튼 사용
* **전체 화면**: 지도를 전체 화면으로 표시

### 이동 컨트롤

* **이동**: 클릭 후 드래그하여 지도 이동

***

## 사용 사례

### 비행 경로 시각화

* 드론 촬영 세션의 커버리지 영역 확인
* 이미지 커버리지의 공백 식별
* 비행 경로 실행 검증

### 지상 조사 검토

* 지상 기반 촬영의 공간적 분포 확인
* 조사 영역 대비 보정 대상 이미지 위치 파악
* 추가 촬영 위치 계획

### 품질 관리

* 예상치 못한 위치에서 촬영된 이미지 신속 식별
* 데이터셋 전체 GPS 정확도 검증
* 현장 기록과 이미지 위치 교차 참조

## 문제 해결

### 마커 표시되지 않음

**가능한 원인:**

* 이미지에 GPS 메타데이터 미포함
* 촬영 중 카메라 GPS 비활성화
* 외부 소프트웨어로 EXIF 데이터 제거

**해결 방법:** 카메라에서 GPS가 활성화되었는지 확인하고 원본 파일을 재수입하십시오

### 잘못된 위치의 마커

**가능한 원인:**

* 카메라 GPS의 위성 신호 수신 불량
* 촬영 중 GPS 드리프트 발생

**해결 방법:** 일반적으로 촬영 시점의 문제입니다. 정밀한 작업에는 PPK/RTK GPS 사용을 고려하십시오
