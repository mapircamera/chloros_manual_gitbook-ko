# 프로젝트

Chloros를 사용하면 나중에 다시 열 수 있는 프로젝트를 만들 수 있습니다.

## 새 프로젝트

<figure><img src=".gitbook/assets/project_new.JPG" alt=""><figcaption></figcaption></figure>

메인 메뉴에서 "새 프로젝트"를 선택하고 프로젝트의 고유한 이름을 입력하세요.

## 프로젝트 열기

<figure><img src=".gitbook/assets/project_open.JPG" alt=""><figcaption></figcaption></figure>

프로젝트 폴더에 있는 기존 프로젝트 목록을 보려면 "프로젝트 열기"를 선택하세요. 프로젝트가 없으면 보조 사이드 메뉴가 열리지 않습니다. 위 사진에는 GUI 생성 프로젝트(t1, t2, t3)가 나열되어 있는 것을 볼 수 있습니다. DATE\_TIME 프로젝트는 기본 프로젝트 명명 체계를 사용하여 CLI에서 생성되었습니다. 프로젝트 이름을 클릭하면 열립니다.

"프로젝트 폴더 열기" 버튼을 클릭하면 프로젝트 경로에서 컴퓨터의 파일 탐색기가 열립니다. [프로젝트 설정](project-settings/page-2.md)에서 프로젝트 경로를 조정할 수 있습니다.
