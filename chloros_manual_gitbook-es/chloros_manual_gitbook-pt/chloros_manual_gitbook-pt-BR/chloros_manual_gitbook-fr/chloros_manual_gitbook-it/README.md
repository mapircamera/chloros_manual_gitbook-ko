# Chloros 사용 설명서 - 이탈리아어

이것은 Chloros Professional 다중 스펙트럼 이미지 처리 소프트웨어 사용자 설명서의 이탈리아어 번역본입니다.

## 클로로스 소개

Chloros는 강력한 데스크톱 소프트웨어, 명령줄 도구 및 Python API를 사용하여 이미지를 실행 가능한 통찰력으로 변환하는 전문 다중 스펙트럼 이미지 처리 소프트웨어입니다.

### 특징
- 데스크탑 및 브라우저 인터페이스
- CLI 및 Python SDK(Chloros+)
- 28개 언어로 이용 가능
- 24개 이상의 식물 지수
- 반사율 교정
- 주변광 센서 지원

## 선적 서류 비치

이 매뉴얼은 GitBook과 동기화되어 있습니다. 문서에는 다음이 포함됩니다.
- 설치 및 설정
- 이미지 처리 워크플로우
- 교정 절차
- 식생 지수 계산
- CLI 및 Python SDK 사용
- API 참조
