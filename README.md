# 시작하기

<div data-full-width="false"><figure><img src=".gitbook/assets/chloros_logo_transparent.png" alt=""><figcaption></figcaption></figure></div>

Chloros는 [MAPIR](https://www.mapir.camera)에서 제공하는 이미지 및 기타 센서 데이터 처리용 소프트웨어 애플리케이션입니다.

***

Chloros는 4가지 애플리케이션 모드로 제공됩니다:

## Chloros: 데스크톱 GUI 애플리케이션

\[권장] 모든 기능을 갖춘 독립형 별도 창.

## Chloros (브라우저): 웹 기반 GUI 인터페이스

웹 브라우저(Chrome/Edge)에서 실행됩니다. 4GB 미만 VRAM을 가진 느린 컴퓨터에 최적입니다.

## [Chloros CLI: 명령줄 인터페이스](CLI.md)

명령줄 기반 배치 처리. 자동화, 스크립팅 및 고급 워크플로우에 최적입니다. _CLI 사용 시 Chloros+ 라이선스가 필요합니다._

## [Chloros API: Python SDK](api-python-sdk.md)

자동화 및 맞춤형 워크플로우를 위한 프로그래매틱 Python 인터페이스. 연구 파이프라인 구축, 기존 Python 애플리케이션 연동, 맞춤형 도구 개발에 이상적입니다. _API 사용 시 Chloros+ 라이선스가 필요합니다._

***

## Chloros+

Chloros는 대부분의 작업에 무료로 사용 가능하지만, 더 많은 기능이 필요할 수 있습니다. 이때 유료 라이선스인 Chloros+가 유용합니다. Chloros+ 라이선스를 통해 다음과 같은 새로운 기능을 사용할 수 있습니다:

* **멀티스레드 처리**: 파이프라인을 통해 이미지를 동시에 처리하여 대규모 프로젝트의 이미지 처리 속도를 크게 향상시킵니다.
* **GPU(CUDA) 가속화**: 최신 GPU 메모리 옵션을 활용하여 이미지 처리 파이프라인을 더욱 가속화합니다. 최적의 결과를 위해 4GB 이상의 VRAM을 권장합니다.
* **Chloros+** [**CLI**](CLI.md) **액세스**: 명령줄에서 Chloros+를 실행하여 자동화하고 자체 소프트웨어에 통합할 수 있습니다.
* **Chloros+** [**API**](api-python-sdk.md) **액세스:** Python에서 Chloros+를 실행하여 프로그래밍 방식으로 제어함으로써 연구 파이프라인, 데이터 분석 워크플로우 및 맞춤형 애플리케이션과의 원활한 통합을 가능하게 합니다.
* **다중 기기 사용**: 각 Chloros+ 라이선스로 2대 이상의 기기를 등록할 수 있습니다. 등록된 기기는 MAPIR Cloud 계정으로 관리하세요. Chloros+ 라이선스를 업그레이드하여 더 많은 기기를 지원할 수 있습니다.
* **사용자 정의 다중 스펙트럼 지수 공식:** Chloros 래스터 계산기에 사용자 정의 다중 스펙트럼 지수를 입력하여 처리 및 이미지 뷰잉 샌드박스 환경에서 활용할 수 있습니다.
