# 출력 이미지 형식

### TIFF (16비트)

16비트 (픽셀 디지털 번호 0 - 65535) TIFF (.tif) 형식 출력.

### TIFF (32비트, 백분율 %)

32비트 (픽셀 부동소수점 0.0 - 1.0) TIFF (.tif) 형식 출력. 픽셀 값은 반사율 백분율을 나타내며, 0.0 = 0%, 1.0 = 100%입니다.

### **JPG (8비트)**

8비트(픽셀 디지털 번호 0 -255) JPG (.jpg) 형식 출력.

### **PNG (8비트)**

8비트(픽셀 디지털 번호 0 - 255) PNG (.png) 형식 출력. 웹 보기 및 시각화에 적합한 무손실 압축.
